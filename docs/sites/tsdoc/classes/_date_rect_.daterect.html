draw

  • draw(): DateRect

  • Overwrites Base.draw

    • Defined in date-rect.ts:78

    This method defines which other methods to call in order to create a punchcard graph with the date on the horizontal axis and the time of day on the vertical axis. Mostly calls methods of the parent class, Base, but for example the symbols for this class are SVG rects, so it calls its own method .drawSymbols() for that.

    Successful drawing depends on whether the container is currently visible, and whether there is enough information in the instance to draw anything at all.

    This method overrides stub method in parent class.

    Returns DateRect

    A reference to an instance of DateRect.
