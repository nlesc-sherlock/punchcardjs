getColor

  • getColor(at: number): [number, number, number, number]
    • Defined in colormap.ts:312

    Returns an rgba color array based on the current colormap (including the domain values at the upper and lower boundaries of the ColorMap) and an input domain value. The color values are determined by linear interpolation between the known color values, as defined by the ColorMap.

    Parameters

    • at: number

      Value for which you want to know the associated color

    Returns [number, number, number, number]

    4-D vector containing rgba value of the color, with 0 represetning zero intensity and 255 full intensity.
